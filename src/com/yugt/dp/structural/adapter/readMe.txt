*定义:
将两个不兼容的类纠合在一起使用，使用Adapter，在这两种接口之间创建一个混合接口


*角色：
Ø 源角色Adaptee：需要适配的目标类或接口。
Ø 目标角色Target：所期望得到的接口。
Ø 适配器角色Adapter：用来把源接口转换成目标接口，显然这一角色不可以是接口，而必须是具体类


*
接口适配器模式的关键在于：对接口抽象化
类适配器模式的关键在于：对类进行适配
对象适配器模式的关键在于：对对象进行包装


*
1.对象的适配器模式是各种模式的起源



*
应用场合：
1.AWT事件适配器（接口的适配器模式）
2.（类的适配器模式）
3.I/O字节流到字符流的适配器（对象的适配器模式）（一层一层封装，增加功能）
